jbpmmigration by eschabell

http://sourceforge.net/apps/trac/jbpmmigration/

The goal of this project is to provide for existing users of jBPM some migration tooling to assist with moving to jBPM5. There will be a phases approach taken to this tooling project as follows: * provide for jPDL3 to BPMN2 ProcessDefinitionMigration (XML) o CLI tool o get it included in JBoss Tools (like right mouse on process definition to convert) * provide for tooling to map jBPM3 API usage to jBPM5 API usage o scan a code base or project o provide suggested mappings o provide list of non-mappable calls or calls needing human intervention * provide jPDL4 to BLPMN2 process definition migration (XML) o same as initial tooling, just extended for jBPM4 * provide for tooling to map jBPM4 API usage to jBPM5 API usage o same as initial tooling, just extended for jBPM4 All of this projects development work will be based on provided use cases. If you have use cases that we are unable to correctly process please feel free to submit!
